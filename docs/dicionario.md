
# Dicionário de dados (v0)

## Território e população
- `geo_id`: identificador (UF/região/município).
- `geo_name`: nome do território.
- `level`: nível territorial (`UF`, `regiao_saude`, `municipio`).
- `pop_target`: população-alvo.

## Necessidades (anual)
- `screened`, `positives`, `colpos`, `biopsy`, `pathology_slides`, `treatment`.

## Capacidade (recursos requeridos)
- `req_colposcopistas`, `req_citotecnicos`, `req_patologistas`, `req_equip_trat`.
